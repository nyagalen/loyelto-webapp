# loyelto-webapp
Le code frontend de l'application web loyel.to
Écrit en Typescript et React avec les librairies tanstack, react-router, MUI. Utilise Privy pour le OAuth

L'appli ne se lance pas avec npm run dev, parce que je ne peux pas push le .env dans la repo, mais ce qu'on voit dans le code est précisément ce qui a été déployé sur loyel.to

Pour accéder à l'authentification Privy: aller sur l'url /business-main
